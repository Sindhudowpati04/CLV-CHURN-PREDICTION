# CLV-CHURN-PREDICTION
Customer Lifetime Value (CLV) and Churn Prediction Using Data Science

Project Overview

This project focuses on predicting Customer Lifetime Value (CLV) and Customer Churn using Data Science and Machine Learning techniques. The objective is to analyze customer behavior and identify customers who may churn while estimating their lifetime value.

Datasets Used

- customers.csv
- transactions.csv
- events.csv
- support_tickets.csv
- churn_labels.csv
- clv_labels.csv

Tools and Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Google Colab
- GitHub

Project Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Feature Engineering
6. CLV Prediction using Regression
7. Churn Prediction using Classification
8. Model Evaluation

Visualizations

- Customer Data Analysis
- Monthly Fee Distribution
- Customer Insights Graphs

Machine Learning Models

- Linear Regression (CLV Prediction)
- Logistic Regression (Churn Prediction)

Results

The models were trained and evaluated successfully. Customer Lifetime Value and Churn Probability were predicted using customer-related features.

Conclusion

This project demonstrates how Data Science can be used to analyze customer behavior, estimate customer value, and predict churn. These insights can help businesses improve customer retention and revenue growth.
